# Git  
  
- 初始化資料夾  
```
git init
```
- 加載    
```
git add .   
```  
- 創建節點    
```
git commit -m "describe"   
```
- 推上git   
``` 
git push {連結名稱} {分支名稱}
``` 
- 若無法推上(要求先pull)  可使用   
```
git push {連結名稱} {分支名稱} --force   
```
- 創建SSH KEY  
```
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```
